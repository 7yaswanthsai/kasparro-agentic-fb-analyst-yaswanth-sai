# Data Directory

This directory contains data files used by the Kasparro Agentic FB Analyst.

## Files

- **sample_fb_ads.csv**: Sample Facebook advertisement data for analysis
- **synthetic_fb_ads_undergarments.csv**: Facebook advertisement data for analysis

## Data Format

The sample_fb_ads.csv file contains columns for:
- campaign_name
- adset_name
- date
- spend
- impressions
- clicks
- ctr
- purchases
- revenue
- roas
- creative_type
- creative_message
- audience_type
- platform
- country


The synthetic_fb_ads_undergarments.csv file contains columns for:
- campaign_name
- adset_name
- datespend
- impressions
- clicks
- ctr
- purchases
- revenue
- roas
- creative_type
- creative_message
- audience_type
- platform
- country
